HTML Table Reference

Table Tags

Tag

Name

Description

<caption>

Caption

The caption or title of a table

<col>

Column

A column within a table

<colgroup>

Column Group

A group of columns within a table

<table>

Table

The wrapper element for all HTML tables

<thead>

Table Head

A set of rows defining the column headers in a table

<th>

Table Head Cell

A cell as header of a group of table cells

<tbody>

Table Body

The set of rows containing actual table data

<tr>

Table Row

A row of cells in a table

<td>

Table Data

A cell of a table that contains data

<tfoot>

Table Foot

The set of rows summarizing the columns of a table

Table Attributes

Attribute

Name

Description

border

Table/Cell Border

For defining a border for the table or individual cells

border-collapse

Collapsed Borders

For collapsing two borders into one

border-spacing

Border Spacing

To set the spacing between cells

colspan

Column Span

Defines how many columns a td element should span

id

ID

To uniquely define one table

padding

Cell Padding

Specifies the space between the cell content and its borders

text-align

Text Align

To align the cell text
